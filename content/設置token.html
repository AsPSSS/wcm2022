設置token

先到github裡，點Developer settings，再點Personal access tokens，接著再點generate new token，然後就設名字，我設的是0624，再點選repo，接著在點選generate token，然後會跑出一段亂碼，複製那串亂碼，打開資料夾wcm2022，找到.git，若找不到就在檢視裡勾選顯示隱藏，接著點開git.找到config，再點開config，接著用記事本開啟，接著把那串亂碼貼在https://的後面，在加上一個@，接著到cmd裡，打git add .，在打git config --global user.email "學校的gmail帳號"在打Git commit -m “0624”，最後再打git push，結束。
